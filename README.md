# Pac-Man Game in Assembly Language

This repository contains a Pac-Man game developed in Assembly Language using the Irvine library. The project demonstrates advanced programming concepts in assembly, including graphics rendering, input handling, and file management, while maintaining optimal performance and user interactivity.

---

## Features

### Core Gameplay
- Interactive gameplay with Pac-Man navigating a maze.
- Dynamic ghost AI with responsive movement.
- Collectible dots that disappear upon collection.
- Lives counter and scoring system.

### Advanced Enhancements
- Complex maze layouts with hidden shortcuts and teleportation paths.
- Bonus items (e.g., fruits) that grant additional points.
- Dynamic difficulty adjustments with increased speed and additional ghosts.
- End screen displaying the player's username and final score.

### User Experience
- Menu navigation and pause functionality.
- Instructions screen for player guidance.
- Sound effects for a more engaging experience.

### File Management
- Player names, scores, and levels are recorded and sorted for easy reference.

---

## Technologies Used

- **Language**: Assembly Language
- **Library**: Irvine32
- **Assembler**: MASM (Microsoft Macro Assembler)
